# v3 Roadmap

## What v3 Adds

This pack covers the topics that usually make beginners stall:

- exception flow
- switch-based logic
- Kotlin-generated bytecode
- reflection-heavy code
- multi-dex layouts
- dynamic loading patterns
- native library awareness

## Best Way To Use v3

1. keep `handbook_v2/02_Smali_Cheat_Sheet.md` open
2. read one v3 chapter at a time
3. find the same pattern in one safe APK
4. explain it in plain English

## Mindset

At this stage, the question is no longer:

- "What does this single line mean?"

Now the question becomes:

- "What larger behavior pattern is this code implementing?"
